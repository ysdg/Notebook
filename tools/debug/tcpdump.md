# 入门

1. 抓取命令：`tcpdump tcp port <portNum> -w <name>`；
2. 使用wireshark查看；

## 网络抓包工具

| 工具    | 说明                                                                                   | 教程 |
| ------- | -------------------------------------------------------------------------------------- | ---- |
| Tshark  | 支持多种协议；<br />Wireshark的命令行版本，兼容tcpdump；<br />支持命令行的展示和分析； |      |
| ngrep   | 轻量、简单的网络文本搜索工具；                                                         |      |
| tcpdump | 抓包工具，需要Wireshark分析；                                                          |      |

参考文献：[Tshark vs TCDump and ngrep Difference | Learning Tree](https://www.learningtree.com/blog/how-do-tshark-ngrep-and-tcpdump-differ-and-when-to-use-them/)
