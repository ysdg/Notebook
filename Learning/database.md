# 数据库

## OLTP vs OLAP

| 分类 | 全称                                           | 说明                                           |
| ---- | ---------------------------------------------- | ---------------------------------------------- |
| OLTP | OnLine Transaction Processsing<br>联机事务处理 | 注重对业务的快速响应和支持，进行增删改查操作。 |
| OLAP | OnLine Analytical Processsing<br/>联机分析处理 | 注重面向应用、市场的数据分析，辅助决策         |
